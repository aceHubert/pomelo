## runtime环境变量
| 变量名       |  描述       | 默认值   |
| ------------ |   -------- | -------- |
| ENV_FILE        |  使用的.env文件       | 开发环境使用 .env.development<br/> 生产环境使用 .env         |  
| CONFIG_FILE        |  读取文件配置，以 dist 为根目录      |  空，根据环境变量转换成配置文件        |
